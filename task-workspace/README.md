#  Task Workspace

A task and time management web application built using **HTML, CSS, and Vanilla JavaScript**.



#  Features

- Add tasks
- Start / pause task timer
- Mark tasks as completed
- Edit and delete tasks
- Light / Dark mode
- Task summary (Total / Completed / Pending)
